# Microsoft Edge Add-ons 上架准备

## 建议标题

LingoTrans - AI 中英文本翻译

## 简短描述

使用你自己的 OpenAI 兼容 API，在侧栏中自动进行中英文本互译。

## 详细描述

LingoTrans 是一个轻量级 AI 中英文本翻译扩展。安装后，用户可以配置自己的 OpenAI 兼容 API Key、Base URL 和模型，然后在扩展侧栏或弹窗中输入文本。输入英文时自动翻译为中文，输入中文时自动翻译为英文。

主要功能：

- 中英文本自动互译
- 支持 Edge 侧栏和工具栏弹窗
- 支持复制译文
- 支持用户自定义 Base URL、模型和 API Key
- 支持获取模型列表、测试连接和一键清除账号数据

## 权限用途

- `storage`：保存用户设置。API Key 只存储在浏览器本机。
- `sidePanel`：在 Edge 侧栏显示翻译器。
- `<all_urls>`：访问用户配置的 OpenAI 兼容接口。由于 Base URL 由用户配置，上架包需要允许可配置的请求地址。

## 隐私表单建议

是否收集个人数据：否。

是否传输文本到第三方服务：是，只有在用户输入文本并触发翻译时，文本会发送到用户配置的 OpenAI 兼容 API。扩展开发者不接收、不存储这些文本。

是否存储账号凭据：API Key 仅存储在用户本机浏览器扩展存储中，用户可随时清除。

## 上架前检查清单

- 已在扩展设置页点击“清除账号数据”。
- 项目中不存在个人 API Key 或私有账号信息。
- `manifest.json` 版本号已更新。
- `PRIVACY.md` 内容已同步到商店隐私说明。
- 图标、截图和描述文字已准备好。
