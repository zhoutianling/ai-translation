# LingoTrans 隐私说明

LingoTrans 不收集、不出售、不共享用户个人数据。

## 本地存储

扩展会把以下配置保存在用户浏览器本机的扩展存储中：

- API Key
- Base URL
- 模型名称

用户可以在设置页点击“清除账号数据”，删除已保存的 API Key。

## 网络请求

当用户输入文本并触发翻译时，扩展会把这些文本发送到用户配置的 OpenAI 兼容 API 地址，用于生成译文。扩展本身不经过自有服务器转发请求。

## 不收集的数据

扩展不会收集浏览历史、身份信息、付款信息、精确位置、联系人，也不会把 API Key 写入扩展包或上传到第三方统计服务。
